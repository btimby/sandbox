Content sandboxing

HTML Content

SmartFile links should not be able to host HTML as the main page of a website. Instead, a browser loading HTML from a SmartFile link should display the markup as text.

Hot linking

Hot linking images can be allowed or disallowed by emitting different headers. Hot linking can also be whitelisted for particular 3-rd party sites.

Embedding HTML via an iframe should be allowed only from certain 3rd party sites.

Iframe embedding

Embedding HTML in an iframe should be allowed only from certain 3rd party sites.
